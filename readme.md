### **Видео 1. Воспоминание духов.**

Пиратское побоище, одного из них случайно убивают. Пираты пытаются воскресить товарища, для этого вызывают духов. Один из них случайно толкает другого, и кости разлетаются. Одна из костей летит-летит-летит, залетает в пианино в лицее, крышка захлопывается. ОГ ночью идет играть на пианино, открывает, а там кость, которую он забирает себе.

### **Видео 2. Пираты**

Пираты находят карту и плывут в лицей, потом ходят по Королеву. Приходят к лицею, но охранница их не пускает. Они выходят, переодевают одного из своих в лицеиста и отправляют.
